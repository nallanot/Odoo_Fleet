# Resource Booking

Module Odoo pour gérer la réservation de ressources (équipements, véhicules, IT), avec workflow d'approbation conditionnelle et check-in/check-out via QR code.

## Fonctionnalités
- Gestion des ressources avec politique d'approbation par ressource.
- Réservations avec vue Liste, Formulaire, Calendrier, Kanban, Graphique et Pivot.
- Validation conditionnelle par gestionnaire de ressource ou supérieur immédiat.
- QR Code pour check-in et check-out.
- Sécurité par groupes (utilisateur / manager).
